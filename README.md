## Installation
Create Python virtual environment
```
python -m venv venv
```
> Then [activate it depending on your os](https://docs.python.org/3/library/venv.html#how-venvs-work)

Install python modules
```
pip install -r requirements.txt
```

## Usage
Test it !
```
python main.py
```
